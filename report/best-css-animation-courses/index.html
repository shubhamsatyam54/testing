सीएसएस एनिमेशन क्या हैं?

CSS एनिमेशन कैस्केडिंग स्टाइल शीट्स के लिए एक प्रस्तावित मॉड्यूल है जो CSS का उपयोग करके HTML दस्तावेज़ तत्वों के एनीमेशन की अनुमति देता है।

जब आप जावास्क्रिप्ट का उपयोग करके एनिमेशन बना सकते हैं तो आपको CSS एनिमेशन क्यों सीखना चाहिए?

कई कारणों से CSS एनिमेशन को JavaScript से अधिक पसंद किया जाता है जैसे:

  1. प्रदर्शन: सीएसएस एनिमेशन आमतौर पर जावास्क्रिप्ट एनिमेशन की तुलना में आसान और तेज़ होते हैं क्योंकि वे हार्डवेयर-त्वरित होते हैं और मुख्य जावास्क्रिप्ट थ्रेड के बजाय ब्राउज़र के कंपोज़िटर थ्रेड पर चलते हैं।
  2. सादगी: सीएसएस एनिमेशन आमतौर पर जावास्क्रिप्ट एनिमेशन की तुलना में लागू करने और बनाए रखने में आसान होते हैं।
  3. अभिगम्यता: CSS एनिमेशन को CSS से नियंत्रित किया जा सकता है और सहायक तकनीकों जैसे स्क्रीन रीडर्स के साथ उपयोगकर्ताओं के लिए सुलभ हैं।
  4. ब्राउज़र समर्थन: सीएसएस एनिमेशन के पास अच्छा ब्राउज़र समर्थन है, जिसका अर्थ है कि वे अधिकांश आधुनिक ब्राउज़रों पर काम करते हैं।

तो, चलिए CSS के साथ कुछ एनिमेशन सीखते हैं।

सर्वोत्तम पाठ्यक्रम मार्गदर्शिका पद्धति

मैंने इस रैंकिंग को पिछले सर्वोत्तम पाठ्यक्रम मार्गदर्शिकाओं में उपयोग की गई अब आजमाई हुई कार्यप्रणाली के बाद बनाया है ( आप उन सभी को यहाँ पा सकते हैं ).

  1. शोध करना: मैंने क्लास सेंट्रल के डेटाबेस का लाभ उठाकर शुरुआत की 100K ऑनलाइन पाठ्यक्रम और 200K+ समीक्षाएं।
  2. मूल्यांकन करना: मैं यह समझने के लिए क्लास सेंट्रल, रेडिट और कोर्स प्रोवाइडर्स पर समीक्षाओं को पढ़ता हूं कि दूसरे शिक्षार्थी प्रत्येक कोर्स के बारे में क्या सोचते हैं और इसे एक के रूप में अपने अनुभव के साथ जोड़ते हैं। सिखाने वाला .
  3. चुनना: अच्छी तरह से बनाए गए पाठ्यक्रमों को चुना गया था यदि वे मूल्यवान और आकर्षक सामग्री प्रस्तुत करते हैं और उन्हें मानदंडों के एक सेट में फिट होना है और तदनुसार रैंक किया जाना है: व्यापक पाठ्यक्रम, सामर्थ्य, रिलीज की तारीख, रेटिंग और नामांकन।

आप क्या सीखेंगे

आप घुमाना, स्लाइड करना, साधारण वस्तुओं को बदलना और यहां तक ​​कि अंतरिक्ष में रॉकेट भेजना भी सीखेंगे।

आप कैसे सीखेंगे

यह स्क्रिम्बा द्वारा पेश किया जाने वाला एक प्रो कोर्स है। जेड खलीली के साथ सीएसएस एनिमेशन सीखें यूट्यूब पर फ्री में उपलब्ध है।

पाठ्यक्रम में बदलाव जोड़ने और उन्हें अनुकूलित करने के साथ-साथ एनिमेशन को परिभाषित करने और समय को नियंत्रित करने के लिए विभिन्न तकनीकों को शामिल किया गया है।

आप क्या सीखेंगे

यह कोर्स आपको बदलाव सिखाएगा, बदलाव को अनुकूलित करेगा और वास्तविक दुनिया की संक्रमण चुनौती पर काम करेगा।
